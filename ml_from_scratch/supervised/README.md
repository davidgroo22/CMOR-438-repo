# Supervised Learning

The first type of algorithms we will cover, which are the majority of this project, are supervised learning algorithms. Supervised learning algorithms are given data points and their labels to train on. They then adjust differently depending on the algorithm but they utilize the labels to decide on what adjustments to make. Below are the supervised learning algorithms covered in this project and a basic explanation of their functions:

- **Perceptron** — a linear binary classifier
- **Linear Regression** — predicts a continuous value via gradient descent
- **Logistic Regression** — binary classification using a continuous exponential function
- **Multilayer Perceptron** — a network of perceptrons built into layers that is able to learn complex patterns
- **K-Nearest Neighbors** — predicts a value based on the data closest to it
- **Decision Tree** — finds the best metrics to classify the data until it is fully classified
- **Random Forest** — builds multiple decision trees and averages their votes

Each of these algorithms has a corresponding README discussing the math behind the algorithm, and a Jupyter notebook demonstrating the algorithm on a dataset, all contained in the `notebooks` directory.
