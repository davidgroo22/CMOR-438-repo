# Unsupervised Learning

Unsupervised learning algorithms are given data points with no labels and hope to group the data based on certain patterns that the algorithm is able to pick up on. They do not necessarily learn like supervised models, but instead they find patterns within the data. Below is the single unsupervised learning algorithm that is covered in this project and a basic description of its function

**K-Means-Clustering** - finds areas where there are "clumps" of data and algorithmically determines the best centers for these clumps of data. Is able to predict new data based on where the nearest clump is.

This algorithm will be explained mathematically and demonstrated in a Jupyter Notebook inside its folder in the `notebooks` directory.
